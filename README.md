# JobTrack – Frontend

## ¿Qué es?
Frontend estático hecho con HTML, CSS y JavaScript.

## ¿Por qué es estático?
Porque no necesita servidor propio.
Se publica directamente en GitHub Pages y consume un backend externo.

## ¿Qué hace?
- Muestra la interfaz de usuario
- Captura acciones del usuario
- Envía solicitudes HTTP al backend
- Renderiza las respuestas

## ¿Cómo se conecta al backend?
A través de llamadas fetch a la API desplegada en Render.
